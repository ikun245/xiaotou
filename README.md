# 落地页前端借鉴GUI

这是一个基于 Python、Tkinter 和 Playwright 的可视化网页采集工具。它模拟真实的浏览器行为，支持自定义 User-Agent，可以加载动态渲染的 H5 页面。

## 功能特点

- **可视化界面**：操作简单，无需编写代码。
- **真实浏览器内核**：使用 Chromium 内核，完美支持 JavaScript 和 H5 渲染。
- **自定义 UA**：支持切换 PC、Mac、iPhone、Android 等多种 User-Agent。
- **源码保存**：一键保存当前渲染后的页面 HTML 源码。

## 安装说明

1. 确保已安装 Python 3.10 或更高版本。
2. 安装依赖库：
   ```bash
   pip install -r requirements.txt
   ```
3. 安装 Playwright 浏览器内核：
   ```bash
   playwright install chromium
   ```

## 使用方法

1. 运行程序：
   ```bash
   python crawler_app.py
   ```
2. 在界面中输入目标网址。
3. 选择想要模拟的 User-Agent。
4. 点击"启动浏览器"。
5. 浏览器窗口弹出后，点击"访问页面"加载网址。
6. 页面加载完成后，点击"保存页面源码"即可将当前页面内容保存到本地。
